# Cursivis 0.1.0 Beta 1

Cursivis brings OpenAI to selected text and screen regions through a compact,
cursor-first Windows interaction. This beta includes Smart Mode, dynamic Guided
Mode, Prompt Optimizer, image analysis, Live Mode with OpenAI Realtime, and a
confirmation-gated browser Take Action system.

## Install

1. Download `Cursivis-Setup-x64.exe` from this release.
2. Open it and approve the normal Windows security prompts.
3. Complete the first-launch Settings checklist and save/test your own OpenAI
   API key.
4. Load the included Chromium extension manually if you want browser Take
   Action support; the installer does not bypass browser extension protections.

Windows Defender SmartScreen may show **Unknown Publisher** because this beta
is not yet code-signed. The installer is self-contained for 64-bit Windows 10
version 2004 and later. The API key is encrypted for the current Windows user
and stored locally; no API key is bundled in the installer.
